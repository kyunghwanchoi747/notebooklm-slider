# NotebookLM Canvas Editor 사용 가이드

본 확장프로그램은 노트북LM의 이미지를 개별적으로 선택하고 삭제할 수 있는 편집기로 변환해줍니다.

## 설치 방법

1.  Chrome 브라우저에서 `chrome://extensions`에 접속합니다.
2.  오른쪽 상단의 **'개발자 모드'**를 활성화합니다.
3.  왼쪽 상단의 **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 클릭합니다.
4.  본 프로젝트 폴더인 `노트북LM 슬라이더 변환` 폴더를 선택합니다.

## 사용 방법

1.  노트북LM 페이지([notebooklm.google.com](https://notebooklm.google.com))를 엽니다.
2.  상단 확장프로그램 바에서 **'NotebookLM Canvas Editor'** 아이콘을 클릭합니다.
3.  **'페이지 이미지 스캔'** 버튼을 누릅니다.
4.  하단에 나타나는 이미지 목록 중 편집하고 싶은 이미지를 클릭합니다.
5.  새 탭에서 **캔버스 에디터**가 열립니다.
    - **객체 선택**: 클릭하여 선택
    - **객체 삭제**: `Del` 키 또는 사이드바의 삭제 버튼
    - **객체 이동**: 마우스 드래그
    - **저장**: PNG 또는 SVG 다운로드

## 기술적 안내

- **로컬 처리**: 모든 이미지는 외부 서버로 전송되지 않고 사용자의 브라우저 내에서 직접 처리됩니다.
- **오픈소스 활용**: [Fabric.js](http://fabricjs.com/)를 사용하여 캔버스 편집 기능을 구현했습니다.
- **VTracer**: 현재 이미지 벡터화 엔진이 로컬에 포함되어 있습니다 (WASM 기반).

> [!NOTE]
> 일부 복잡한 이미지는 벡터화 과정에서 시간이 걸릴 수 있습니다.
